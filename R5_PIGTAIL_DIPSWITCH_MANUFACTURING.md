## What changed from R4

R5 removes the risky board-mounted OBD2/J1962 connector footprint and replaces it with robust pigtail/cable solder pads.

R5 also replaces solder-jumper routing selectors with DIP switches:

- `SW1`: CAN normal/cross routing selector
- `SW2`: LIN/K single-wire source selector

This makes the board easier to configure without soldering/desoldering jumpers.

## Mechanical approach

Instead of mounting an OBD connector directly to the PCB:

1. Use an OBD2 male pigtail cable or OBD2 male connector with cable.
2. Solder each OBD wire to the matching labeled PCB pad.
3. Use the two strain-relief holes near the cable entry with a zip tie.
4. Put the OBD connector at the vehicle end of the cable.

This avoids the previous mechanical blocker: no exact board-mounted OBD connector datasheet was available.

## Teensy 4.1 pin audit

The Teensy footprint used in R5 was audited against the PJRC Teensy 4.1 pinout card.

Main assignments:

| Function | Teensy pin |
| --- | --- |
| CAN A TX / CTX1 | 22 |
| CAN A RX / CRX1 | 23 |
| CAN B TX / CTX2 | 1 |
| CAN B RX / CRX2 | 0 |
| LIN/K UART TX | 8 |
| LIN/K UART RX | 7 |
| CAN A standby/control | 5 |
| CAN B standby/control | 4 |
| LIN enable/control | 6 |
| VIN from board 5 V rail | VIN |
| Ground | GND |

Important PJRC facts used:

- Main side headers: two long rows, 24 pins each.
- Header pitch: 2.54 mm.
- Row spacing: 15.24 mm / 0.6 inch.
- PJRC hole guidance: 0.965 mm finished hole diameter for the main through-hole locations.
- Teensy 4.1 I/O is not 5 V tolerant.

## DIP switch configuration

### SW1: CAN normal/cross selector

`SW1` is an 8-position SPST DIP switch.

| SW1 position | Function |
| --- | --- |
| 1 | CAN A normal: OBD6 -> CAN_A_H |
| 2 | CAN A normal: OBD14 -> CAN_A_L |
| 3 | CAN A crossed: OBD6 -> CAN_A_L |
| 4 | CAN A crossed: OBD14 -> CAN_A_H |
| 5 | CAN B normal: OBD3 -> CAN_B_H |
| 6 | CAN B normal: OBD11 -> CAN_B_L |
| 7 | CAN B crossed: OBD3 -> CAN_B_L |
| 8 | CAN B crossed: OBD11 -> CAN_B_H |

Safe defaults:

- For standard OBD CAN on pins 6/14: turn ON SW1-1 and SW1-2 only.
- For CAN A crossed test: turn ON SW1-3 and SW1-4 only.
- For CAN B normal candidate: turn ON SW1-5 and SW1-6 only.
- For CAN B crossed candidate: turn ON SW1-7 and SW1-8 only.
- Never enable normal and crossed for the same CAN bus at the same time.

### SW2: LIN/K source selector

`SW2` is an 8-position SPST DIP switch.

| SW2 position | Source connected to LIN/K transceiver bus |
| --- | --- |
| 1 | OBD7 |
| 2 | OBD15 |
| 3 | OBD1 |
| 4 | OBD8 |
| 5 | OBD9 |
| 6 | OBD12 |
| 7 | OBD13 |
| 8 | AUX_LIN_K_IO |

Safe defaults:

- Turn ON exactly one SW2 position at a time.
- Typical legacy diagnostic test: SW2-1 ON for OBD7.
- Older/variant L-line test: SW2-2 ON for OBD15.
- Leave all SW2 switches OFF until the target vehicle pin is known.

## Before connecting to a vehicle

1. Inspect the manufactured PCB visually.
2. Check for solder bridges with a magnifier.
3. With no Teensy installed, power from current-limited bench supply first.
4. Verify +5 V rail before installing Teensy.
5. Confirm OBD cable wiring with a multimeter pin-by-pin.
6. Confirm SW1/SW2 settings before plugging into the car.
7. Start with passive/read-only firmware.
8. Do not enable CAN termination unless you intentionally need it.

## Manufacturing notes

Recommended PCB fab settings:

- 2-layer FR4
- 1.6 mm thickness
- 1 oz copper
- HASL or ENIG acceptable
- Solder mask: any color
- Silkscreen: white recommended

The Gerber ZIP was generated by KiCad CLI from the DRC-clean routed board. It is suitable for PCB fabrication upload.

Assembly still requires correct component procurement and soldering. The PCB files do not purchase parts automatically.
